o-checkbox

creates a wrapper and uniform, no-frills checkbox. Note the jade example, this structure is a must.

Example

Sass

.checkbox
  +o--(o-checkbox) // using o-'s silent @extend mixin

Jade

.checkbox
  input#my-checkbox(type='checkbox')  
  label(for='my-checkbox') Check Me

based on this pen

o-loader

a pure css (and atypical) animated loader; implemented as a mixin to keep keyframes out of the final output if the loader is not being used.

o--

shorthand for silent @extend (do not pass % with $name)

o-x

shorthand for normal @extend (do not pass . with $name)
