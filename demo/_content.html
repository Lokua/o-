%o-v-center-parent

Container for vertically centering child elements. To comply, child elements must have vertical-align: middle; or you can simply @extend %o-v-center-child. Note that %o-center-parent also has vertical-align: middle, so parents can be nested.

see %o-v-center-child

%o-checkbox

Creates a wrapper and uniform, no-frills checkbox. Note the jade example, this structure (including the label) is a must.

Example

Sass

.checkbox
  +o--(o-checkbox) // using o-'s silent @extend mixin

Jade

.checkbox
  input#my-checkbox(type='checkbox')  
  label(for='my-checkbox') Check Me

based on this pen

%o-grid

Generic grid container with 100% width and clearfix extension

=o-span($number)

Dynamically add $number of columns to an element. The o-grid is based on a 12 column grid with no gutters, By default, a media query is included to set width to 100%, or 12 columns on screens with 768px max which can be changed through the $o-max-width-break variable. To disable the break functionality entirely, set $o-auto-break: false.

%o-col-*

%o-col-* placeholders, where * is the number of columns from 1-12 see also =o-span

=o-generate-grid-classes

Use this mixin if you'd prefer to generate static bootstrap-style .o-grid and o-col-* (.o-col-1 through .o-col-12) classes.

=o-loader

a pure css (and atypical) animated loader; implemented as a mixin to keep keyframes out of the final output if the loader is not being used.

o-as-percent($number)

Simply add a % symbol to a unit (does not perform decimal conversion).

o-golden-ratio($number: 100, $scale: 1)

get the golden ratio of $number recursively scaled $scale times. A scale setting less then 1 will return the smaller portion.

Example

o-golden-ratio(30,  1) //=> 48.54102
o-golden-ratio(30,  2) //=> 78.54102
o-golden-ratio(30, -1) //=> 18.54102
o-golden-ratio(30, -2) //=> 11.45898
